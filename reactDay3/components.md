# Day 03 - React Learning

## Topics Covered

### State

State is used to store and manage dynamic data inside a component.

Features:
- Dynamic UI Updates
- Re-rendering Components
- User Interaction Handling

### Components

Components are reusable building blocks of a React application.

Types:
1. Functional Components
2. Class Components
